Tutorial: Creando un Juego con IA

Cómo construimos "Adivina el Refrán" paso a paso usando Claude Code

1

Preparar el contenido en Excel

Primero creamos un archivo Excel (idiomas.xlsx) con 50 frases hechas y refranes populares en tres idiomas.

Prompt utilizado:

Busca las 50 frases hechas que más se dicen en español y tradúcelas al inglés y francés. Ponlas en un Excel con columnas: ESPAÑOL, INGLES, FRANCES.

Herramientas usadas:

Python openpyxl
Resultado: 50 frases como "A quien madruga, Dios le ayuda" con sus equivalentes en inglés ("The early bird catches the worm") y francés.
2

Crear la página de inicio

Diseñamos una página de bienvenida atractiva con animaciones CSS.

Elementos incluidos:

  • Fondo degradado oscuro con partículas flotantes
  • Título con efecto de brillo (glow)
  • Banderas animadas de UK, Francia y España
  • Instrucción para comenzar (clic o tecla)

Técnicas CSS utilizadas:

@keyframes glow { from { text-shadow: 0 0 20px rgba(255,255,255,0.5); } to { text-shadow: 0 0 40px rgba(255,255,255,0.8); } }
3

Desarrollar la lógica del juego

El juego muestra frases en inglés o francés y el usuario debe escribir el equivalente en español.

Funcionalidades implementadas:

  • Selección aleatoria de frases e idioma
  • Marcador de aciertos, racha y progreso
  • Mensajes de felicitación o ánimo
  • Puntuación final al completar las 50 frases
Reto técnico: ¿Cómo aceptar respuestas que no sean exactamente iguales pero sean correctas?
4

Implementar el algoritmo de similitud

Usamos el algoritmo de Levenshtein para calcular la similitud entre la respuesta del usuario y la correcta.

function similarity(str1, str2) { const maxLen = Math.max(str1.length, str2.length); const distance = levenshteinDistance(str1, str2); return (maxLen - distance) / maxLen; } // Aceptamos si la similitud es >= 80% if (similarity(userAnswer, correctAnswer) >= 0.8) { // ¡Correcto! }

Esto permite que pequeños errores ortográficos no penalicen al jugador.

5

Añadir traducción con IA (Anthropic API)

Integramos la API de Claude para traducir las frases a otros idiomas en tiempo real.

Frontend

HTML + JavaScript

Backend

Python Server

IA

Claude API

Servidor Python (server.py):

client = anthropic.Anthropic(api_key=API_KEY) message = client.messages.create( model="claude-sonnet-4-20250514", max_tokens=200, messages=[{ "role": "user", "content": f"Traduce esta frase al {idioma}: {frase}" }] )
Importante: La API key nunca debe subirse a repositorios públicos. Por eso creamos un archivo .gitignore que excluye server.py.
6

Publicar en GitHub Pages

Subimos el proyecto a GitHub y lo publicamos con GitHub Pages para que cualquiera pueda jugar.

Pasos realizados:

  1. Crear repositorio en GitHub
  2. Inicializar git local: git init
  3. Añadir archivos: git add index.html juego.html idiomas.xlsx
  4. Commit: git commit -m "Juego Adivina el Refrán"
  5. Conectar con remoto: git remote add origin [URL]
  6. Subir: git push -u origin main
  7. Activar GitHub Pages en Settings > Pages
Resultado: Juego disponible en una URL pública accesible desde cualquier navegador.
